# Projeto (API) 

O objetivo deste projeto é desenvolver uma plataforma de Business Inteligence no Power BI, que permita visualizar e analisar a eficiência dos terminais portuários brasileiros, com base em dados reais providos pelo site da Agência Nacional de Transportes Aquaviários (ANTAQ).

# Objetivo do Projeto
Este projeto tem como objetivo ajudar e facilitar na utilização da plataforma GitHub, visando:
* Centralizar os trabalhos e projetos;
* Organizar e estruturar as informações;
* Versionar e controlar as alterações;
* Facilitar o compartilhamento e feedback;
* Desenvolver habilidades técnicas.



## Tecnologias Utilizadas

* Power BI
* Microsoft Excel
* Python (Colab)
* R.
* GitHub
